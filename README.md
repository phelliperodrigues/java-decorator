# Design Patterns
## Decorator

This Project Implementation the Decorator Design Patterns

#### En
The function of the Design Pattern Decorator is to "decorate" an existing behavior.

In my example, we decorate taxes to receive other taxes from their builders, so we create an abstract class that is responsible for calculating the incorporated tax.

#### Pt

A Função do Design Pattern Decorator é "decorar" um comportamento existente.

No meu exemplo, decoramos os impostos para receberem em seu construtores outros impostos, assim um imposto pode ser composto de outros impostos, então criamos uma classe abstrada que tem como responsabilidade calcular o imposto incorporado.
